ID e CLASS

ID

O ID funciona como um CPF. Asiim como só posso ter um numero de CPF, cada elemento só pode ter um valor de ID. O CPF é unico, ou seja, ninguem pode ter um CPF igual ao meu. isso significa que o ID tambem é unico, entao nao é possivel repetir o mesmo valor 2 vezes.

CLASS

O CLASS funciona de maneira oposta do ID. Diferente do ID em que voce so pode ter um valor, no CLASS voce pode ter varios valores. Tambem diferentes do ID, em que nao se pode repetir um valor, no CLASS voce pode utilizar o mesmo valor para varios elementos
